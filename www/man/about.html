:: Fantomas Iptconf: Описание и назначение программы ::


    Fantomas Iptconf manager - открытая front-end система для управления, учета и контроля сетевого трафика Iptables/Netfilter, использующая для работы только штатные средства пакета Netfilter (iptables, ipset, ulogd), без дополнительных программ.

    Программа представляет собой Open-source/PHP менеджер, состоящий из веб-интерфейса и небольшого консольного CLI-скрипта. Основное назначение и цель программы - создать и предоставить конечному пользователю наиболее удобные и легкие в использовании инструменты для эффективного использования широких встроенных возможностей пакета Netfilter framework (iptables, ipset, ulogd). Программа имеет средства для управления правилами доступа, подсчета трафика, ведения базы статистики и многое другое...
    Это если очень кратко...

    Лицензия: программа Fantomas Iptconf manager является бесплатной и распространяется по свободной лицензии. Вы можете свободно использовать программу в любых как коммерческих так и личных целях, кроме целей продажи или перепродажи самой программы третьим лицам.
